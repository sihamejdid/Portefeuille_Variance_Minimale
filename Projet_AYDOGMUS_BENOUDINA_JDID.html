2. Mtriques pour le portefeuille quipondr

Les mtriques calcules pour ce portefeuille incluent :

  • Le rendement attendu \(Er = \sum_{i=1}^n{w_i\cdot\mu_i}\) avec \(w_i\) les poids et \(\mu_i\) les rendements moyens.
  • La variance \((\sigma^2)\)
  • Lcart-type StDev
  • Le ratio de Sharpe : mesure la performance ajuste au risque, soit le rapport entre le rendement moyen et lcart-type
  • La VaR (Value at Risk) \(5\%\) : indicateur du reste de perte extre un niveau de confiance de 95% selon une loi normale.
  • La concentration : indice donnant la concentration du portefeuille en faisant la somme des carrs des parts des secteurs cest--dire \(\sum_{j}{(x_j)^2}\). Plus le portefeuille est diversifi, plus sa valeur tend vers 0.
  • Lquivalent Certain \((EC)\) : diffrence entre le rendement moyen du portefeuille et un multiple de la variance du portefeille dpendant dun coefficient daversion au risque : \(EC = E[r_p] - \frac{\gamma}{2}\sigma^2(r_p)\). O \(\gamma = 3\) le coefficient daversion quon dfinit pour la suite de notre projet. Plus ce coefficient est grand, plus lindividu est prodent.

Cela permet dtablir un tableau de synthse qui fournit une base pour comparer cette stratgie des approches plus complexes.

B. Portefeuille de variance minimale

1. Matrice de covariance

Il est maintenant question de mettre en oeuvre une optimisation pour construire un portefeuille de variance minimale. Ici, les poids des titres sont ajusts de manire minimiser la variance totale du portefeuille, en utilisant la matrice de covariance pour quilibrer les relations entre les rendements. Cette stratgie est particulirement utile pour les investisseurs souhaitant rduire leur exposition au risque global. Contrairement au portefeuille quipondr, ce portefeuille tient compte des interactions complexes entre les titres pour atteindre une rpartition des fonds optimale en termes de risque.

Le problme doptimisation quadratique est rsolu laide de la bibliothque quadprog. Lobjectif est de minimiser la variance sous deux contraintes principales :

  • La somme des poids doit tre gale 1 : \(\sum_{i=1}^n{w_i=1}\)
  • Les poids doivent tre positifs : \(w_i \geq 0\)

En dautres termes, voici le programme rsoudre : \[\left\{\begin{array}{c} min_{x_j} \ \sigma^2(r_p) \\ \sum_{j}{x_j} = 1 \\ \forall j,x_j \geq 0 \end{array} \right.\]

Autrement dit, on va chercher minimiser la variance en fixant les parts \(x_j\) sous la double contrainte que la somme des parts doit tre gale 1 (contrainte budgaire) et que chaque part va tre positive ou nulle (interdiction de la vente dcourvert).

La solution obtenue ajuste les poids de manire maximiser la diversification tout en minimisant lexposition au risque.

# tout en respectant la contrainte que la somme des poids = 100%.

# Fonction pour optimiser le portefeuille
optimize_portfolio <- function(mean_returns, cov_matrix) {
  n <- length(mean_returns)  # Nombre de titres
  Dmat <- cov_matrix  # Matrice des variances-covariances
  dvec <- rep(0, n)  # Vecteur nul pour minimisation
  Amat <- cbind(rep(1, n), diag(n))  # Contraintes : somme des poids = 1 et poids >= 0
  bvec <- c(1, rep(0, n))  # Valeurs des contraintes
  result <- solve.QP(Dmat, dvec, Amat, bvec, meq = 1)  # Rsolution de l'optimisation
  result$solution  # Retourne les poids optimaux
}

  # Calcul du rendement attendu
  portfolio_return <- sum(weights * mean_returns)
  
  # Calcul de la variance et de la volatilit
  portfolio_variance <- t(weights) %*% cov_matrix %*% weights
  portfolio_volatility <- sqrt(portfolio_variance)
  
  # Calcul du ratio de Sharpe (annualis)
  sharpe_ratio <- (portfolio_return * 12) / (portfolio_volatility * sqrt(12))
  
  # Calcul de la VaR  5% (annualise)

    return = portfolio_return * 12,  # Rendement annualis
    volatility = portfolio_volatility * sqrt(12),  # Volatilit annualise
    sharpe = sharpe_ratio,  # Ratio de Sharpe annualis

  ) %>%

    Portfolio = recode(  # Remplace les valeurs de la colonne 'Portfolio' pour les rendre plus lisibles

      equip = "Equip",  # Remplace 'equip' par 'Equip'

    ),
    Metric = case_when(  # Remplace les valeurs de la colonne 'Metric' en fonction de mots-cls
      grepl("EC", Metric) ~ "EC",  # Si 'Metric' contient "EC", le nom devient "EC"
      grepl("Sharpe", Metric) ~ "Sharpe",  # Si 'Metric' contient "Sharpe", le nom devient "Sharpe"
      grepl("Er", Metric) ~ "Er",  # Si 'Metric' contient "Er", le nom devient "Er"
      grepl("Vol", Metric) ~ "Vol",  # Si 'Metric' contient "Vol", le nom devient "Vol"

plot_metric <- function(metric_name, y_label, is_percentage = FALSE) {  # Dclaration de la fonction avec trois arguments : 'metric_name', 'y_label' et un argument optionnel 'is_percentage'
  # Filtrage des donnes pour la mtrique spcifie

    filter(Metric == metric_name)  # Applique un filtre sur la colonne 'Metric' pour slectionner uniquement les lignes correspondant  'metric_name'
  # Cration du graphique avec ggplot
  ggplot(metric_data, aes(x = start_date, y = value, color = Portfolio)) +  # Initialise ggplot avec 'start_date' sur l'axe x, 'value' sur l'axe y, et 'Portfolio' comme couleur
    geom_line(size = 1) +  # Ajoute une ligne pour chaque srie, avec une paisseur de ligne de 1
    labs(  # Ajoute des labels au graphique
      title = metric_name,  # Le titre du graphique est dfini par le nom de la mtrique
      x = "Date",  # Label de l'axe x
      y = y_label  # Label de l'axe y (pass en argument  la fonction)
    ) +
    scale_y_continuous(  # Modification de l'chelle de l'axe y
      labels = if (is_percentage) scales::percent_format(scale = 1) else scales::label_number()  # Si 'is_percentage' est TRUE, les valeurs sont formates en pourcentage, sinon elles sont affiches sous forme numrique
    ) +
    theme_minimal() +  # Applique un thme minimaliste pour le graphique
    theme(  # Personnalisation du thme
      legend.title = element_blank(),  # Supprime le titre de la lgende
      legend.position = "bottom",  # Place la lgende en bas du graphique
      plot.title = element_text(size = 14, face = "bold", hjust = 0.5)  # Modifie le titre du graphique pour le rendre en gras, de taille 14, et centr horizontalement
    )
}
